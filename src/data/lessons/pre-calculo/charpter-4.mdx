# Intervalos e Operações com Conjuntos

Imagine que um engenheiro diz: "Esta temperatura deve estar entre 20°C e 30°C". Não é só 20 ou só 30 — é **tudo entre eles**. Ou, ao contrário: "Qualquer valor menor que -10 ou maior que 50 causará falha".

Os **intervalos** são subconjuntos especiais de ℝ que descrevem essas "faixas" contínuas. E as **operações entre conjuntos** — união, interseção, complementar — permitem combinar essas faixas de forma poderosa.

Nesta lição, vamos dominar a linguagem e as operações de conjuntos que você usará em todo o Cálculo.

## Objetivos de Aprendizado

Ao final desta lição, você será capaz de:

- **Definir** e **notacionar** intervalos (abertos, fechados, semi-abertos)
- **Representar** intervalos na reta numérica
- **Converter** entre notação de conjunto e notação de intervalo
- **Realizar** operações entre conjuntos (união, interseção, complementar)
- **Interpretar** desigualdades como intervalos
- **Combinar** intervalos usando operações de conjunto
- **Aplicar** intervalos em problemas de domínio e restrições

## Tipos de Intervalos

Um **intervalo** é um conjunto contínuo de números reais. Existem diversos tipos, dependendo se os extremos estão incluídos ou não.

### 1. Intervalo Fechado [a, b]

Um intervalo **fechado** inclui os dois extremos:

$$[a, b] = \{x \in \mathbb{R} : a \leq x \leq b\}$$

- Notação: **[colchete quadrado significa "incluído"**
- Na reta: segmento com os dois extremos fechados
- Exemplos:
  - $[0, 1]$ inclui 0, 1 e tudo entre
  - $[-3, 5]$ inclui -3, 5 e tudo entre

### 2. Intervalo Aberto (a, b)

Um intervalo **aberto** exclui os dois extremos:

$$(a, b) = \{x \in \mathbb{R} : a < x < b\}$$

- Notação: **(parêntese significa "excluído"**
- Na reta: segmento com os dois extremos abertos
- Exemplos:
  - $(0, 1)$ inclui 0.5, 0.9, 0.999... mas **não** 0 nem 1
  - $(-3, 5)$ não inclui -3 nem 5

### 3. Intervalos Semi-Abertos

Um intervalo pode incluir um extremo e excluir o outro:

- **[a, b)** inclui $a$, exclui $b$: $\{x : a \leq x < b\}$
  - Na reta: segmento com extremo esquerdo fechado e direito aberto

- **(a, b]** exclui $a$, inclui $b$: $\{x : a < x \leq b\}$
  - Na reta: segmento com extremo esquerdo aberto e direito fechado

### 4. Intervalos Infinitos

Quando um intervalo se estende infinitamente, usamos o símbolo $\infty$:

### Exemplo 2: Converter Desigualdade em Intervalo

Converta a desigualdade em notação de intervalo:

1. $-2 < x \leq 5$ → $(-2, 5]$
2. $x \geq 0$ → $[0, \infty)$
3. $x < -1$ → $(-\infty, -1)$
4. $|x| < 3$ (ou $-3 < x < 3$) → $(-3, 3)$

## Operações com Conjuntos

Quando você trabalha com intervalos, frequentemente precisa **combinar** ou **intersecar** eles. As operações fundamentais são:

<IntervalSetOperationsLab />

### 1. União (∪)

A **união** de dois conjuntos $A$ e $B$ inclui **todos os elementos que estão em $A$ ou em $B$** (ou em ambos):

$$A \cup B = \{x : x \in A \text{ ou } x \in B\}$$

Na reta numérica, a união é a **reunião de todos os pontos**:

<IntervalWorkedExampleDiagram variant="union-13-25" />

Resultado: $A \cup B = [1, 5)$

### 2. Interseção (∩)

A **interseção** de dois conjuntos $A$ e $B$ inclui **apenas os elementos que estão em ambos**:

$$A \cap B = \{x : x \in A \text{ e } x \in B\}$$

Na reta numérica, a interseção é a **sobreposição**:

<IntervalWorkedExampleDiagram variant="inter-13-25" />

Resultado: $A \cap B = (2, 3]$

### 3. Complementar (A' ou A^c)

O **complementar** de um conjunto $A$ em relação a ℝ inclui **todos os números reais que não estão em $A$**:

$$A^c = \{x \in \mathbb{R} : x \notin A\}$$

Se $A = [1, 3]$, então:

$$A^c = (-\infty, 1) \cup (3, \infty)$$

## Diferença entre Conjuntos

A **diferença** $A \setminus B$ (lê-se "$A$ menos $B$") inclui elementos que estão em $A$ mas **não** em $B$:

$$A \setminus B = \{x : x \in A \text{ e } x \notin B\}$$

## Aplicações: Domínio de Funções

Os intervalos são **essenciais** para especificar o **domínio** de uma função.

<FunctionDomainVisualizer />

### Exemplo 6: Domínio de Uma Função

Encontre o domínio de $f(x) = \frac{1}{x - 2}$.

O denominador não pode ser zero:
$$x - 2 \neq 0$$
$$x \neq 2$$

**Domínio:** Todos os reais exceto 2

$$\text{Dom}(f) = (-\infty, 2) \cup (2, \infty)$$

Ou em notação de conjunto: $\{x \in \mathbb{R} : x \neq 2\}$

### Exemplo 7: Domínio com Raiz Quadrada

Encontre o domínio de $g(x) = \sqrt{x - 3}$.

Raiz quadrada precisa de um radicando não-negativo:
$$x - 3 \geq 0$$
$$x \geq 3$$

**Domínio:** $[3, \infty)$

## Propriedades das Operações

As operações entre conjuntos seguem leis, similares à aritmética:

<MarkdownTable
  headers={["Propriedade", "Descrição", "Fórmula"]}
  rows={[
    ["**Comutativa**", "Ordem não importa", "$A \\cup B = B \\cup A$; $A \\cap B = B \\cap A$"],
    ["**Associativa**", "Agrupamento não importa", "$(A \\cup B) \\cup C = A \\cup (B \\cup C)$"],
    ["**Distributiva**", "Pode \"distribuir\"", "$A \\cap (B \\cup C) = (A \\cap B) \\cup (A \\cap C)$"],
    ["**De Morgan**", "Complemento de operação", "$(A \\cup B)^c = A^c \\cap B^c$"],
    ["**De Morgan**", "Complemento de operação", "$(A \\cap B)^c = A^c \\cup B^c$"],
  ]}
/>

### Exemplo 8: Lei de De Morgan

Verifique $(A \cup B)^c = A^c \cap B^c$ com $A = [0, 2]$ e $B = (1, 3)$.

**Lado esquerdo:**
$$A \cup B = [0, 3)$$
$$(A \cup B)^c = (-\infty, 0) \cup [3, \infty)$$

**Lado direito:**
$$A^c = (-\infty, 0) \cup (2, \infty)$$
$$B^c = (-\infty, 1] \cup [3, \infty)$$
$$A^c \cap B^c = [(-\infty, 0) \cup (2, \infty)] \cap [(-\infty, 1] \cup [3, \infty)]$$
$$= (-\infty, 0) \cup [3, \infty)$$

**Ambos resultam em:** $(-\infty, 0) \cup [3, \infty)$ ✓
